   Visit: https://makersuite.google.com/app/apikey
   - Sign in with Google
   - Click "Create API Key"
   - Copy the key

4. CONFIGURE .ENV:
   - Open the .env file
   - Find: GEMINI_API_KEY=your_gemini_api_key_here
   - Replace with your actual API key
   - Save the file

5. START APPLICATION:
   python app.py

6. OPEN IN BROWSER:
   http://127.0.0.1:5000

7. CREATE BUSINESS PROFILE:
   - Click "Set Up Your Business"
   - Enter your business details
   - Click "Create Profile"

8. GENERATE CONTENT:
   - Go to "Generate" page
   - Click "Generate Post"
   - Copy the generated content

🚨 COMMON ISSUES & SOLUTIONS

ISSUE: Python not found
SOLUTION: Install Python from https://www.python.org/downloads/
         Make sure to check "Add Python to PATH"

ISSUE: Module not found errors
SOLUTION: pip install -r requirements.txt

ISSUE: Port 5000 already in use
SOLUTION: Edit .env and change FLASK_PORT to 5001

ISSUE: Gemini API errors
SOLUTION: Check .env has correct API key
         Restart Flask app
         Check internet connection

ISSUE: Database errors
SOLUTION: Delete database.db file
         Restart app (auto-creates fresh database)

# AI Social Media Automation

Flask-based AI social media automation platform using Python and Gemini AI.

## Features
- AI content generation
- Caption generation
- Flask dashboard
- Content history
- Business setup
- Automation-ready architecture

## Tech Stack
- Python
- Flask
- Gemini AI
- HTML/CSS
- GitHub

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Run App

Open browser:

```text
http://127.0.0.1:5000
```
